# SRS - Сценарии использования

**Теги:** #SRS #usecases #testing

**Основные сценарии:**

1. **Прохождение уровня:**
   - Игрок начинает уровень.
   - Игрок использует движение и прыжки для перемещения по платформам.
   - При встрече с препятствием (летящий снаряд) игрок активирует замедление времени, чтобы увернуться.
   - Игрок доходит до конца уровня и собирает осколок кристалла.

2. **Смерть игрока:**
   - Игрок падает в пропасть или сталкивается с врагом.
   - Игра показывает экран смерти ("Game Over").
   - Игрок может перезапустить уровень с последней контрольной точки.

3. **Использование рычага:**
   - Игрок подходит к рычагу.
   - Нажимает клавишу взаимодействия (`E`).
   - Включается анимация и открывается проход к следующей секции.
