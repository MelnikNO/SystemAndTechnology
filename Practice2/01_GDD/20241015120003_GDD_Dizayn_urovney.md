# GDD - Дизайн уровней

**Теги:** #GDD #levels #design

**Принципы проектирования:**
1. **Постепенное усложнение:** Уровни знакомят с одной механикой (замедление), затем комбинируют её с другими (движущиеся платформы, враги).
2. **Обучение через геймплей:** Игрок учится использовать замедление на простых препятствиях (падающие камни) перед тем, как столкнуться с врагами.
3. **Несколько путей:** Уровни должны иметь скрытые области, доступные только при использовании замедления в определенное время.

**Ключевые элементы:**
- Платформы, исчезающие со временем.
- Снаряды, летящие по сложной траектории.
- Головоломки с синхронизацией движения объектов.
