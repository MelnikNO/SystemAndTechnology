# IDEA - Улучшения механик

**Теги:** #IDEA #mechanics #time #innovation

**Новые концепции для улучшения:**

1. **"Ускорение времени":**
   - Альтернативная способность (активируется другой клавишей), которая ускоряет время для игрока, позволяя быстрее перемещаться по прямым участкам или разгонять снаряды, чтобы они пробивали стены.

2. **"Временные копии":**
   - При активации способности создается призрачная копия игрока, которая повторяет его действия последние 2 секунды. Это позволяет нажимать несколько кнопок/рычагов одновременно.

3. **"Парадокс":**
   - Сложные уровни, где игрок должен использовать "замедление" и "ускорение" одновременно для решения головоломок (например, замедлить одну платформу, но ускорить другую).
