# SDD - Модуль управления персонажем

**Теги:** #SDD #player #controls

**Класс `Player`:**

**Атрибуты:**
- `x, y` (координаты)
- `speed_x, speed_y` (скорость)
- `is_jumping`, `is_grounded` (состояние)
- `time_energy` (шкала времени)
- `sprite` (изображение)

**Методы:**
- `handle_input()` -> Считывает нажатия и обновляет скорость.
- `update()` -> Обновляет позицию, применяет гравитацию и коллизии.
- `apply_time_dilation(factor)` -> Изменяет скорость всех объектов в зависимости от фактора замедления.
- `collect_item(item)` -> Обрабатывает сбор предметов.

**Взаимодействие:**
- Использует `PhysicsManager` для проверки коллизий.
- Использует `TimeManager` для применения эффекта замедления.
