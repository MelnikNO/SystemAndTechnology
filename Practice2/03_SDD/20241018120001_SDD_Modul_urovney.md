# SDD - Модуль уровней

**Теги:** #SDD #levels #modules

**Описание модуля:**
- Отвечает за загрузку данных уровней из файлов (JSON или CSV).
- Создает объекты платформ, врагов и предметов на основе данных.
- Управляет переключением между уровнями.

**Интерфейс:**
- `load_level(level_id)` -> Возвращает список объектов для рендеринга.
- `get_spawn_point()` -> Возвращает координаты старта игрока.
- `next_level()` -> Переключает на следующий уровень.

**Технические детали:**
- Уровни хранятся в папке `data/levels/` в виде текстовых файлов.
- Формат файла: JSON с массивами для каждого типа объектов.
- Используется `pygame.sprite.Group()` для оптимизации рендеринга.
