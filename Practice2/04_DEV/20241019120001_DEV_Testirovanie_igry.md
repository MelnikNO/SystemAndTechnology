# DEV - Тестирование игры

**Теги:** #DEV #testing #qa

**Методики тестирования:**

1. **Юнит-тесты:**
   - Проверка физики (гравитация, коллизии) на небольших изолированных сценариях.
   - Тестирование логики AI врагов.
   - Проверка корректности загрузки уровней.

2. **Интеграционные тесты:**
   - Взаимодействие игрока с платформами и врагами в полном уровне.
   - Проверка переключения между состояниями (меню -> игра -> пауза).

3. **Игровые тесты (Playtesting):**
   - Приглашение 3-5 тестировщиков.
   - Сбор обратной связи по сложности и управлению.
   - Запись геймплея для анализа.

4. **Тестирование производительности:**
   - Измерение FPS на разных уровнях.
   - Проверка времени загрузки.
